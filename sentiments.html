2020 Twitter Sentiments

  • Category: Natural Language Processing
  • Project date: December 2020
  • Project URL: https://github.com/shivam-vohra/2020-Twitter-Senitments

Because of how crazy 2020 has been, individuals must understand how others feels on a topic so as to educate them on those topics. Using TextBlob for natural language processing and the Twitter API in python, tweets were gathered on either pre-set topics or user inputted topics. Then, using matplotlib, data was visualized to show the percentage of positive and negative tweets on those topics.
